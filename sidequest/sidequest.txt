Anthropic / Claude Side Challenge Integration
We integrated Claude Code as an autonomous experimentation layer in our Datathon workflow. Based on an AutoResearch loop, we started from a strong baseline and used Claude Code to repeatedly propose small code changes, run a fixed training/evaluation cycle (~8 minutes), and keep only changes that improved our target metric. We ran two Claude Code agents in parallel on a shared GitHub repo, exchanged the best commits after each loop, and connected each agent to its own Kaggle account to get fast public-score feedback. This made Claude a core part of our model development process, not just a writing assistant, and materially improved iteration speed and final performance.
